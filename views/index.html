Exercise tracker

1). Create a New User

POST /api/exercise/new-user

2). Add exercises

POST /api/exercise/add

3). GET users's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

Example 1: https://exercise-logger-danny.herokuapp.com/api/exercise/log?userId=5f59161d158b283fd83e7cb7

Example 2: https://exercise-logger-danny.herokuapp.com/api/exercise/log?userId=5f59161d158b283fd83e7cb7&from=2018-02-05&limit=3
